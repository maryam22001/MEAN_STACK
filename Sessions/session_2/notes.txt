CSS FLEXBOX QUICK REVISION

1. AXES (The Key to Alignment)
   - Main Axis: The direction items flow (Default: Horizontal/Row).
   - Cross Axis: Perpendicular to Main Axis (Default: Vertical/Column).

2. ALIGNMENT PROPERTIES
   - justify-content: Aligns along the Main Axis (e.g., X-axis) X axis when row , Y axis when column.
    
     * space-between: Items pushed to start/end edges; space goes between them.
     * space-around: Space distributed around each item (visual gap).
   - align-items: Aligns along the Cross Axis (e.g., Y-axis).

3. WRAPPING & SHORTHAND
   - flex-wrap: Allows items to wrap onto new lines if space runs out.
   - flex-flow: Shorthand for "flex-direction" + "flex-wrap".

4. FLEXBOX VS GRID
   - Flexbox: 1-Dimensional (Row OR Column). Best for components and alignment.
   - Grid: 2-Dimensional (Rows AND Columns). Easier to control complex layouts.

5. GRID TEMPLATE & UNITS
   - `fr` (Fractional Unit): Consumes available free space.
     * `1fr 1fr`: Two equal columns.
     * `1fr 2fr`: Second column is twice the width of the first.
   - `auto`: Width is determined by the content inside.
   - `repeat()`: Shorthand to avoid repetition.
     * `repeat(3, 1fr)` = `1fr 1fr 1fr` (3 equal columns).
   - `minmax(min, max)`: Sets a range for size (e.g., `minmax(100px, 1fr)`).

6. GAPS (Gutters)
   - `column-gap`: Space between columns (vertical lines).
   - `row-gap`: Space between rows (horizontal lines).
   - `gap`: Shorthand. `gap: 20px 10px;` (20px row gap, 10px col gap).
7. ANIMATIONS & TRANSITIONS
    - transition: Smoothly animates CSS property changes over time.
      * Syntax: `transition: property duration timing-function delay;`
      * Example: `transition: all 0.3s ease-in-out;`
    
    - @keyframes: Defines custom animations with multiple stages.
      * Define keyframes (0%, 50%, 100%) and CSS properties for each.
      * Apply with `animation: name duration timing-function delay iteration-count;`
      * Example:
         @keyframes slideIn {
            0% { transform: translateX(-100%); }
            100% { transform: translateX(0); }
         }
         .element { animation: slideIn 1s ease-in; }
    
    - Key Properties:
      * animation-duration: How long animation runs.
      * animation-timing-function: Speed curve (ease, linear, cubic-bezier, etc.).
      * animation-delay: Wait time before animation starts.
      * animation-iteration-count: How many times it plays (number or infinite).
      * animation-direction: Forward, backward, alternate, or alternate-reverse.
      * animation-fill-mode: Maintains state before (backwards), after (forwards), or both (both).
    
    - Use Cases:
      * Transitions: Simple property changes (hover effects, button interactions).
      * Keyframes: Complex multi-step animations (loading spinners, page transitions).

8. BOOTSTRAP 5 QUICK NOTES

   - Grid System:
     * Every `.row` is divided into 12 columns.
     * Breakpoints: `sm` (≥576px), `md` (≥768px), `lg` (≥992px), `xl` (≥1200px), `xxl` (≥1400px).
     * Example: `col-md-6` means "take 6 of 12 columns on medium screens and up".

   - Gutters (Gaps):
     * Gutters are the gaps between columns.
     * Control them on the `.row` with `g-*`, `gx-*` (horizontal), or `gy-*` (vertical) classes.
     * `g-0`: Removes gutters. `g-5`: Largest gutter.

   - Offsetting Columns:
     * Use `offset-*` to push columns to the right.
     * Example: `offset-md-2` pushes a column by 2 column widths on medium screens.

   - Spacing Utilities (Margin & Padding):
     * Format: `{property}{sides}-{size}`
     * Property: `m` (margin), `p` (padding).
     * Sides: `t` (top), `b` (bottom), `s` (start/left), `e` (end/right), `x` (horizontal), `y` (vertical).
     * Size Scale (0-5):
       - `0` = 0
       - `1` = .25rem
       - `3` = 1rem
       - `5` = 3rem
