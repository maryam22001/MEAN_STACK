# Session 5- JavaScript Fundamentals

## Arrays methods :
push: adds an element at the end of the array
pop :removes the last element of the array
shift:removes the first element of the array
unshift: adds an element at the beginning of the array

***************************************************************
## Strings methods :
slice: removes and you specify the index of the elemeent you want to remove
indexof: displays the index of an element 
last index of :returns the index of the last occurrence of an element in the array
include : returns true and flase 
sort : sort in alphapitical order 
reverse: reverse the order of the elements in the array
concat : creates a new array by concatenating the original array with the new array 
****************************************************************
## Loops:
while: repeats a block of code as long as a condition is true (unknown number of iterations)
do while: executes a block of code once, then repeats while a condition is true
for: repeats a block of code a specific number of times (known iterations)
for...in: loops through object properties
for...of: loops through array values

****************************************************************
## Function Declaration:
- Function Declaration: named function defined with `function` keyword
- Function Expression: function assigned to a variable
- Arrow Function: concise syntax using `=>` operator

## if vs switch:
- Use `if`: for complex conditions, ranges, or boolean logic
- Use `switch`: for single variable with multiple specific values (cleaner and more readable)
