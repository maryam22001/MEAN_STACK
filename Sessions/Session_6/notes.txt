###Dark mode logic :
Breakdown of the Methods:
classList.toggle("dark"): If the class exists, remove it. If it doesn’t, add it. This is the "Switch."

classList.contains("dark"): This returns a Boolean (true/false). We use this to know which "state" the app is currently in.

innerHTML: This is how we inject FontAwesome icons (HTML strings) into our paragraph or div.

Manipulating Content & Attributes
innerHTML: Changes the text and renders HTML tags.

innerText: Changes only the text (ignores tags).

setAttribute("class", "head"): Manually sets an attribute.

classList.toggle("dark"): The "switch"—adds the class if missing, removes if present.

style.color = "green": Inline CSS styling.
